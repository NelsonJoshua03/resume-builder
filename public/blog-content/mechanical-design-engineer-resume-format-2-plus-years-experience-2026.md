## Resume Formatting Tips

### Do's:
- Use standard font (Calibri, Arial, Times New Roman)
- Keep resume to 2 pages maximum
- Use bullet points for easy readability
- Include quantifiable achievements
- Save as PDF file
- Name file properly: "YourName_MechanicalDesignEngineer.pdf"

### Don'ts:
- Don't use fancy graphics or tables
- Don't use headers/footers
- Don't include photos unless specified
- Don't use colored text
- Don't use unusual fonts
- Don't exceed 2 pages
